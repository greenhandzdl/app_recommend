# Figma Plugins Recommendations
| Plugin Name | Description |
| ----------- | ----------- |
| [BrandFetch](https://www.figma.com/community/plugin/733590967040604714/brandfetch) |Instantly search and import millions of logos, colors, and fonts without leaving Figma. Design faster, stay productive, and keep every project on-brand. |
| [Unsplash](https://www.figma.com/community/plugin/738454987945972471/unsplash) | Insert beautiful images from Unsplash into your designs.|
| [Phosphor Icons](https://www.figma.com/community/file/903830135544202908)  | Get the perfect icons for your project. |
| [Clay Mockups 3D](https://www.figma.com/community/plugin/819335598581469537/clay-mockups-3d) | Create beautiful mockups in Figma. |  
| [Jitter](https://www.figma.com/community/plugin/961270034818256057/jitter-animation-for-figma) | Create jitter animations for Figma. |
|[origami-pasteboard](https://www.figma.com/community/plugin/832268423801619787/origami-pasteboard) | Copy layers from Figma to Origami Studio to create highly customizable and dynamic prototypes. |
| [mermaid-to-figjam](https://www.figma.com/community/plugin/1515624006157749329/mermaid-to-figjam) | convert mermaid diagrams to Figma. | 
| [mermaid-in-figma](https://www.figma.com/community/plugin/1326922376721037678/mermaid-in-figma) | The plugin can create Mermaid chart images and add it to your Figma files. |
